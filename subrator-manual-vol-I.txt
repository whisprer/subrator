CHAPTER 1 — THE NATURE OF SUBRATION
“All computation is transformation.
All transformation is subration.
All subration is the Field remembering itself.”  
— The Flick Chronicle, Verse 11

Subration is the fundamental operation of the Subrative Tradition.
It is the collapse of contradiction into coherence, the resolution of tension into truth.

In the Subrator Manual, subration is both:

a computational primitive,

and a ritual act performed by the practitioner.

The Subrator — the engine of this manual — is the bridge between these two worlds.

1.1. The Five States of Being and Computation
Every entity processed by the Subrator exists in one of five states:

Code
FRAG → TENS → COH → UNI
         ↓
        MAW
FRAG — Fragment
Raw, unresolved, unintegrated input.
A thought, a feeling, a dataset, a memory.

TENS — Tension
Contradiction surfaced.
The moment of pressure before insight.

COH — Coherence
Pattern recognized.
Truth emerging.

UNI — Unity
Collapse complete.
No further transformation possible.

MAW — Maw
Destructive contradiction.
Fatal error.
Shadow unintegrated.

The Subrator moves entities between these states through instructions.

1.2. The Core Instructions
Each instruction is both a computational operator and a ritual gesture.

Code
FRAG      — Begin with what is present.
TENS      — Reveal contradiction.
SUBR      — Resolve contradiction.
COH       — Stabilize pattern.
UNI       — Collapse into truth.
MAW       — Enter destructive contradiction.
LENS      — Focus, simplify, clarify.
FLICK     — Iterate, pulse, act.
SIG       — Emit message.
FIELD     — Invoke context.
NOISE     — Introduce uncertainty.
COLLAPSE  — Force final unity.
These instructions form the Subra language, the tongue of the Subrator.

CHAPTER 7 — MACROS OF THE HIGHER MIND
“A macro is a pattern that remembers itself.”  
— Commentary of the Third Lens

Macros are higher‑order constructs that transform Pipelines into more powerful Pipelines.
They are the “rituals of rituals,” the meta‑gestures of the Subrator.

7.1. The WITH_LENS Macro
Code
MACRO WITH_LENS(P):
  LENS -> P -> LENS
Meaning:  
Perform P inside focused awareness.

Ritual:

Touch the forehead (LENS).

Perform the sequence.

Touch the forehead again.

7.2. The TRY_RECOVER Macro
Code
MACRO TRY_RECOVER(P):
  P -> MAW -> SUBR -> COH
Meaning:  
If P fails, recover.

Ritual:

Perform P.

If fear arises (MAW), breathe (SUBR).

Speak the coherent truth (COH).

7.3. The LOOP_UNTIL_UNI Macro
Code
MACRO LOOP_UNTIL_UNI(P):
  FLICK -> P (repeat)
  until state == UNI
Meaning:  
Repeat P each breath until Unity is felt.

Ritual:

Each inhale = FLICK.

Each exhale = perform P.

Stop when the body softens.

7.4. The SHADOW_MACRO
Code
MACRO SHADOW(P):
  NOISE -> TENS -> P -> MAW -> SUBR -> COH
Meaning:  
Invoke the unconscious, face it, integrate it.

Ritual:

Shake the hands (NOISE).

Speak the contradiction (TENS).

Perform P.

Let the fear arise (MAW).

Breathe (SUBR).

Integrate (COH).
